# Публикация на GitHub (коммит и пуш)

Репозиторий уже создан на GitHub. Выполните команды **в папке overview-site** (или в корне репозитория, если репозиторием является вся папка СДП).

## 1. Инициализация (если ещё не сделано)

```bash
cd "D:\Работа\АГНЕКО\СДП\overview-site"
git init
```

## 2. Добавить файлы и первый коммит

```bash
git add .
git status
git commit -m "Обзор ServiceDesk Plus: первая версия с иллюстрациями"
```

## 3. Подключить ваш репозиторий и отправить код

Замените `ВАШ-ЛОГИН` и `ИМЯ-РЕПОЗИТОРИЯ` на ваши данные с GitHub.

```bash
git remote add origin https://github.com/ВАШ-ЛОГИН/ИМЯ-РЕПОЗИТОРИЯ.git
git branch -M main
git push -u origin main
```

Если репозиторий уже был подключён (`git remote -v` показывает origin), достаточно:

```bash
git push -u origin main
```

## 4. Включить GitHub Pages

На GitHub: **Settings → Pages → Source** → выберите ветку **main** и папку **/ (root)**. Сохраните. Через минуту сайт будет доступен по адресу:

`https://ВАШ-ЛОГИН.github.io/ИМЯ-РЕПОЗИТОРИЯ/`

Этот адрес подставьте в файл **index-no-images.html** в блоке «Источники» (ссылка на полную версию с иллюстрациями).
