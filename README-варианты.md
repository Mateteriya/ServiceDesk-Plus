# Три варианта обзора ServiceDesk Plus

## 1. HTML без картинок (один файл для клиента)
- **Файл:** `index-no-images.html`
- **Назначение:** текстовый обзор для отправки клиенту одним файлом (по запросу «просто обзор текстовый»).
- **Как использовать:** отправить клиенту только файл `index-no-images.html`. Он открывается в любом браузере, картинок и внешних ресурсов нет.

## 2. HTML с картинками (публикация на GitHub)
- **Файл:** `index.html` + папка `images/` со скриншотами.
- **Назначение:** полная версия для публикации (например, GitHub Pages); можно добавлять скриншоты в `images/`.
- **Как использовать:** выложить репозиторий на GitHub, включить GitHub Pages — ссылка будет вида `https://ваш-логин.github.io/имя-репо/`.

## 3. DOCX (красиво оформленный документ)
- **Скрипт:** `generate_docx.py`
- **Зависимости:** `pip install -r requirements-docx.txt` (или `pip install python-docx`)
- **Запуск:** в папке `overview-site` выполнить `python generate_docx.py`
- **Результат:** в этой же папке создаётся файл `ServiceDesk-Plus-обзор.docx` с оформлением (заголовки, таблицы редакций с оттенками, глоссарий, списки).

После генерации DOCX его можно дооформить в Word по вкусу (шрифты, титульная страница).
